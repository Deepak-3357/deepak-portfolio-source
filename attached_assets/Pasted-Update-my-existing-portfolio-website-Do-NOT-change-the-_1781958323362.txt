PROJECT TITLE

Smart Attendance Management System

SHORT DESCRIPTION

AI-powered attendance management desktop application using face recognition to automate attendance marking, analytics, reporting, and administration through a modern Qt 6 interface.

TECHNOLOGIES USED

• C++17
• OpenCV 4.12
• Qt 6 Widgets
• LBPH Face Recognition
• Haar Cascade Classifier
• CSV Database Storage
• Visual Studio 2026
• CMake & Ninja

PROJECT DETAILS MODAL

Problem

Manual attendance management is time-consuming, prone to human errors, and vulnerable to proxy attendance. Educational institutions require an automated and reliable solution for accurate attendance tracking.

Features

• Student Registration and Management
• Face Dataset Collection using Webcam
• LBPH Model Training
• Real-Time Face Detection and Recognition
• Liveness Detection
• Automatic Attendance Marking
• Duplicate Attendance Prevention
• Unknown Face Detection
• Attendance Analytics Dashboard
• Date-Wise Attendance Reports
• Student Profile Viewer
• Email Report Generation
• Admin Authentication
• Modern Qt GUI Interface

How It Works

The system captures live video through a webcam.

Haar Cascade detects faces.

LBPH recognizes registered students.

Liveness detection verifies genuine presence.

Attendance records are automatically stored in CSV reports.

Attendance statistics and reports are displayed through the Qt dashboard.

Architecture

↓

↓

↓

↓

↓

↓

↓
Qt Dashboard GUI

Challenges Faced

• Varying lighting conditions affected face detection and recognition accuracy.

• Preventing proxy attendance using photos or mobile screens required liveness detection techniques.

Future Scope

• Integrate Deep Learning-based face recognition (FaceNet / ArcFace) for improved accuracy.

• Develop a cloud-enabled web/mobile dashboard with multi-camera support for remote attendance monitoring.

Outcome

Successfully developed a complete AI-powered attendance management system capable of real-time facial recognition, automated attendance logging, attendance analytics, report generation, and GUI-based administration.

The system reduces manual effort, improves attendance accuracy, and enhances security through liveness verification and automated record management.
